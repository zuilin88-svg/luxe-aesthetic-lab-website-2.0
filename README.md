## Luxe Aesthetic Lab Website 2.0

Vite + React + TypeScript marketing site for Luxe Aesthetic Lab.

## Run locally

**Prerequisites:** Node.js


1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`

## Build

`npm run build`
